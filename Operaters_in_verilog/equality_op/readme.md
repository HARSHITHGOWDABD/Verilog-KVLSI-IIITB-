# EQUALITY OPERATOR
```
1. It is also a binary operator
2. Logic equality(==,!=) returns 0,1,x.
3.Case equality (===,!===) returns 0 or 1.
4.Logic equality returns X if either operands has X or Z data.
5.***Case equality comapares all bits including X and Z.
```
# OPERATORS 
```
A) A==B ----> Equality.
B) A!B ------> Inequality.
C) A===B -----> Case equality. // can compare with x and z
D)A!==B ------>Case inequality.
