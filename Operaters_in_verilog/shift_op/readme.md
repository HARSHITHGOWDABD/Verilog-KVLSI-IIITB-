# SHIFT OPERATOR
```
<< --->  LOGIC LEFT SHIFT (OP1>>2)
>> --->  LOGIC RIGHT SHIFT (OP1<<2)
<<< -->  ARITHEMETIC LEFT SHIFT 
>>> -->  ARITHMETIC RIGHT SHIFT
```
NOTE 
```
1.Shift operation is a binary operation .
2.In logic shift the vacent position is filled with zero .
3.Arithmetic shift works on signed data .
4.** Arithemetic right shift pads MSB bits.
5. Arithemtic right shift pads MSB bits
6. shift by 1'bx or 1'bz results in x.
```
